# LARPgen

6 months ago I was mass-applying to PM roles and writing LinkedIn posts nobody read.

Today I'm the founder of LARPgen.

We just launched. Zero funding. Zero employees. Just me, Claude, and an unhealthy amount of conviction.

Here's the thing most people don't realize: every viral tech post follows a formula. The founder vulnerability arc. The VC thesis drop. The 4am grinder manifesto. The AI thought leader panic tweet.

I studied thousands of them. I reverse-engineered the patterns. And I built a tool that generates them on demand.

**LARPgen** is a persona cosplay engine for the chronically online. You tell it what you want to post about, pick your archetype (Founder, VC, Grinder, or AI Thought Leader), choose a platform and tone, and it writes the post for you. Streaming. In real time. Indistinguishable from the real thing.

The tech stack? Next.js. Tailwind. Claude API. No database. Stateless. Deployed on Vercel. I shipped the MVP in one sitting. Some people call that reckless. I call it velocity.

We're pre-revenue, pre-users, pre-everything. But the conviction has never been higher.

This is the tool I wish I had when I was still LARPing manually.

Now everyone can LARP at scale.

```
npm install
npm run dev
```
